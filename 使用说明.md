# 眼底血管图像分割系统 - 快速使用指南

## 🚀 快速开始

### 第一步：安装依赖
双击运行 **`安装依赖.bat`**，等待所有依赖包安装完成。

### 第二步：启动程序
双击运行 **`启动程序.bat`**，系统将自动启动。

### 第三步：登录系统
使用默认管理员账号登录：
- 用户名：`admin123`
- 密码：`admin123`

## 📋 主要功能

### 1. 用户管理
- 注册新用户（账号密码8-18位）
- 修改密码
- 管理员可删除用户

### 2. 图像分割
1. 点击 **"选择图像"** 按钮，选择眼底图像文件
2. 点击 **"开始分割"** 按钮，等待分割完成
3. 查看分割结果
4. 使用滑块可以缩放图像（1%-500%）

### 3. 配置保存路径
点击 **"路径选择"** 按钮，选择分割结果保存目录。
默认保存在 `out_file` 文件夹。

### 4. 清空结果
点击 **"清空"** 按钮，清除当前显示的图像和日志。

## 🎨 界面说明

- **左侧区域**：显示原始眼底图像
- **右侧区域**：显示分割后的血管图像
- **右侧面板**：
  - 状态日志：显示操作记录
  - 控制按钮：选择图像、开始分割、清空
- **顶部菜单**：路径选择、帮助、关于、退出

## ⚙️ 高级功能

### 模型训练
```bash
python train.py
```
使用自己的数据集训练新模型。

### 批量预测
```bash
python predict.py
```
批量处理测试集中的所有图像。

### 性能评估
```bash
python evaluate.py
```
评估模型在测试集上的性能指标。

## 📁 文件说明

- `main.py` - 主程序（GUI界面）
- `checkpoint.pth` - 训练好的模型权重
- `users.json` - 用户数据（自动生成）
- `data/` - 训练和测试数据目录
- `out_file/` - 分割结果保存目录
- `启动程序.bat` - 快速启动脚本
- `安装依赖.bat` - 依赖安装脚本

## ❓ 常见问题

### Q1: 程序无法启动？
A: 确保已运行 `安装依赖.bat` 安装所有依赖包。

### Q2: 分割速度慢？
A: 首次运行会加载模型，需要一些时间。如果有NVIDIA显卡，程序会自动使用GPU加速。

### Q3: 找不到模型文件？
A: 确保 `checkpoint.pth` 文件在程序同一目录下。

### Q4: 支持的图像格式？
A: 支持 .tif, .png, .jpg, .jpeg 等常见格式。
